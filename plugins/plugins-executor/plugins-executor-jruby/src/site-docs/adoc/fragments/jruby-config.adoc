== Configure the JRuby Executor

The JRuby executor is added to the configuration as follows:

[source%nowrap,json]
----
"engineServiceParameters":{
  "engineParameters":{
    "executorParameters":{
      "JRUBY":{
        "parameterClassName" :
        "org.onap.policy.apex.plugins.executor.jruby.JrubyExecutorParameters"
      }
    }
  }
}
----
